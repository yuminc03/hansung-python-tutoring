# 파이썬 튜터링 아카이브 웹사이트 개발 기획서

2026학년도 1학기 미래플러스대학 한성 AX 융합 튜터링의 '파이썬 프로그래밍' 과목을 위한 미니멀하고 애플 스타일의 교육용 아카이브 웹사이트 기획서입니다.

## 1. 개요 및 기술 스택
- **프레임워크**: React 18, Vite (SPA)
- **라우팅**: React Router DOM
- **스타일링**: Vanilla CSS + CSS Variables (디자인 기획서의 디테일한 여백과 색상을 완벽히 통제)
- **기능 패키지**: `react-markdown` (마크다운 파싱), `react-syntax-highlighter` (코드 하이라이팅), `lucide-react` (아이콘)
- **폰트**: Pretendard (애플 스타일의 모던 산세리프)

## 2. 디자인 시스템 (CSS 변수)
- **Background**: `#F9FAFB` (메인 배경), `#FFFFFF` (카드 및 사이드바)
- **Text**: `#111827` (주요 텍스트), `#6B7280` (보조 텍스트)
- **Point**: `#007AFF` (애플 스타일의 은은한 블루, 링크 및 버튼 강조용)
- **Code Block**: `#1E1E1E` (다크 테마 배경)
- **Typography**: `line-height: 1.6 ~ 1.8`로 넉넉한 줄 간격을 확보하여 가독성 극대화.

## 3. 페이지 설계
### [페이지 1] 홈 대시보드 (Home)
- **Hero 섹션**: 한성 AX 융합 튜터링을 명시하는 부제목과 넉넉한 여백 배치.
- **주차별 카드 그리드**: 
  - CSS Grid를 사용하여 반응형 레이아웃 구성.
  - 마우스 호버 시 부드럽게 떠오르는 마이크로 애니메이션(`transform`, `box-shadow`) 적용.
  - 1주차~6주차의 핵심 주제와 요약 내용을 표시.

### [페이지 2] 학습 노트 콘텐츠 (Note)
- **고정형(Sticky) 좌측 사이드바**:
  - 스크롤을 내려도 화면 좌측에 항상 고정.
  - 현재 페이지(마크다운)의 소제목(H2)을 자동으로 파싱한 목차(CONTENTS) 제공.
  - 1~6주차로 빠르게 이동할 수 있는 네비게이션 링크 제공.
- **메인 콘텐츠 (마크다운 렌더링 영역)**:
  - 최대 너비를 800px 내외로 제한하여 긴 글 읽기에 최적화된 화이트 스페이스 유지.
  - 구문 강조(Syntax highlighting)가 적용된 다크 테마 코드 블록 표시.
  - 코드 블록 우측 상단에 원클릭 복사(Copy) 버튼 구현.

## 4. 데이터 및 콘텐츠 관리
- 주차별 실제 학습 내용은 `src/data/notes/` 폴더 내에 마크다운(`.md`) 파일로 작성.
- 코드를 수정하지 않고 마크다운 텍스트만 업데이트해도 웹페이지에 자동으로 반영되도록 구성.
